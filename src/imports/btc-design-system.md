Design a **modern web application UI/UX** for a vocational and language training institution called **Brotherly Training Center (BTC)**.

The platform should combine a **website, student portal, and a modern Learning Management System (LMS)**.

The design must be **very modern, clean, and professional**, similar to platforms like **Coursera, Notion, and Stripe dashboards**. Use **soft gradients, smooth animations, glassmorphism elements, and a minimalist layout**.

Create a **full design system** including:

* Typography scale
* Color palette (primary, secondary, accent)
* Buttons, inputs, cards, tables
* Dashboard components
* Dark mode and light mode

The base name of the platform should be **Brotherly Training Center (BTC)**.

---

### Tech mindset for the design

The UI should be designed with developers in mind and easily implementable using:

* React / Vite
* Supabase (Auth, Database, Storage)
* PostgreSQL
* Drizzle ORM
* Tailwind CSS
* Shadcn UI
* SWR for data fetching
